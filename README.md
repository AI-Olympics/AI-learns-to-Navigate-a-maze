### Maze environment

Run the main.py file.
- Make a maze for the AI
- Then train the agent.
- Save it's policy
- Test it in test.py file

- Or load the pre-trained mazes and AIs

And watch the AI find it's way through your maze.

### Requirements

- OpenCV
- pygame
- numpy
